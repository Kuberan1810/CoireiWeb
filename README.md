# React + TypeScript + Vite

This project uses **React + TypeScript + Vite** for fast development with Hot Module Replacement (HMR).

### Official React Plugins

Choose one of the following:

- `@vitejs/plugin-react` (Babel / OXC)
- `@vitejs/plugin-react-swc` (SWC)

---

# React Compiler

The React Compiler is not enabled by default because of its impact on development and build performance.

If needed, follow the official React documentation to enable it.

---

# ESLint

Enable type-aware linting for production projects.

Recommended configurations:

- `recommendedTypeChecked`
- `strictTypeChecked`
- `stylisticTypeChecked`

Use both:

- eslint-plugin-react-x
- eslint-plugin-react-dom

for improved React linting.
